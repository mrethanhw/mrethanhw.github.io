Artificial Intelligence Projects

Brave Bot

Link to Github Repository

A model of fan attendance at Atlanta Braves baseball games. Using over 15 variables including team win-loss record, weather, date, and player statistics, this model built using Gaussian Process Regression accurately predicts game attendance within 5000 fans.

Data-Driven Models for Covid-19 Policymaking

Link to Github Repository

Comprehensive datasets of state-level policy responses to COVID-19, as well as regression analysis of the effects of economic closures on community transmission and comparison of lockdown stringency between states using K-means clustering. Analyses can be referred to by state governments to enact strategic public-health policies in response to the pandemic.

CS 3600: Introduction to Artificial Intelligence

Demonstration of algorithms for machine learning and graph theory in the classic game PACMAN. Projects done in this course include reinforcement learning for maximizing points scored, conducting Bayesian inference of enemy ghost locations, and implementing A*/BFS/DFS maze traversal.
